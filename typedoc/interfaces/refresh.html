Interface Refresh<T>

A refreshable cache can be refreshed with a request to the API. When calling refresh, all data in the cache will be replaced with the results returned by the API, which means that older data may be lost. On endpoints with support for pagination, this should not be called too often. (Still should be called once in a while to make sure deleted items get purged, since we have no update mechanism for that.)
